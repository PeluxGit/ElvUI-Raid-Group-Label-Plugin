# ElvUI Raid Group Label

Shows your current raid group as a small label on the **ElvUI player frame**.

---

## Highlights

- Displays the player's current raid subgroup.
- Text format options: `#`, `G#`, `Group #`.
- Configurable font size.
- Optional solo display (`0` when not in raid).
- Optional custom background and border colors.
- Supports ElvUI Movers.
- Supports anchor-based positioning (point/relative point/offsets).

---

## Requirements

- World of Warcraft (Retail)
- ElvUI

---

## Configure

ElvUI -> **UnitFrames** -> **Individual Units** -> **Player** -> **Raid Group Label**

Main options:

- **Enable**
- **Text Format**
- **Font Size**
- **Show When Solo**
- **Style**
- **Position** (`Label Point`, `Player Point`, `X/Y Offset`, `Reset Position`)

---

## Movers

- Use ElvUI Movers to drag the label.
- Right-clicking the mover opens this addon's options page.
