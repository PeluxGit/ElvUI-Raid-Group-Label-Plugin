# Changelog

## [2026-02-13]

### Initial release

- Added an ElvUI plugin that displays the player's current raid group on the player unit frame.
- Added text format options: `#`, `G#`, and `Group #`.
- Added configurable font size.
- Added `Show When Solo` option (shows group `0` outside raid).
- Added auto-sizing for the label based on rendered text.
- Added ElvUI mover support with persistent position saving.
- Added anchor-based positioning (`Label Point`, `Player Point`, `X/Y Offset`) and reset action.
- Added style options for custom background and border colors with reset support.
